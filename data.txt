🌍 Global Tools Website – Full Overview (MUI + JSX + Context API)

1. Tech Stack

Framework: Next.js 14 (App Router)

UI: MUI (Material UI)

Icons: MUI Icons

State Management: Context API (theme, language, auth if needed)

i18n: next-intl (or next-intl) for multi-language routing

Dark/Light Mode: Built-in with MUI theme + Context toggle

Fonts: Google Fonts (Roboto → default in MUI, or Inter/Poppins for modern look)

SEO: Next.js Metadata API, sitemap, robots.txt

Ads/Monetization: Google AdSense integration + Affiliate links

🎨 Design System Guide (Next.js + MUI + JSX + Context API)

1. Colors (Global Theme)

Choose clean, professional colors (finance tools need trustworthiness, student tools need clarity, utilities need simplicity, fun tools can use accents).

Light Mode

Primary (Brand): #1976d2 (MUI Blue)

Secondary: #009688 (Teal)

Background: #ffffff

Surface (cards, tool inputs): #f5f5f5

Text Primary: #212121

Text Secondary: #616161

Accent (Fun tools / highlights): #ff9800 (Orange)

Dark Mode

Primary: #90caf9 (Blue 200)

Secondary: #80cbc4 (Teal 200)

Background: #121212

Surface: #1e1e1e

Text Primary: #e0e0e0

Text Secondary: #9e9e9e

Accent: #ffb74d

2. Typography (Global Fonts)

Font Family: Roboto (default in MUI, global readability)

Alternative: Inter or Poppins for modern look

Scale:

H1: 36px / Bold → Tool titles, Blog titles

H2: 28px / Semi-bold → Section headers

H3: 22px → Subsections

Body: 16px → Tool descriptions, content

Small: 14px → Labels, captions

3. Components (MUI + Customization)
Navbar

Layout:

Left → Logo

Center → Links: Tools (dropdown) | Blog | About | Contact

Right → Language Switcher | Theme Toggle | Search (optional)

Design:

Transparent → solid background on scroll

Active link → Primary color underline

Footer

Sections:

Left → Logo + short tagline

Middle → Quick links (Tools, Blog, About, Contact)

Right → Social Icons + Language Selector

Style:

Background → Primary 900 (dark blue) in light mode, black in dark mode

Text → White / Gray

ToolCard

Design:

MUI Card with icon + tool title + short description

Hover → subtle shadow + lift

Click → navigates to tool page

Tool Page Layout

Header: Tool name + intro text (H1 + Body)

Input Section:

Use MUI Grid for responsiveness

Inputs → Sliders, TextFields, Dropdowns (outlined style)

Results Section:

MUI Paper cards (with results: EMI, GPA, etc.)

Typography → H2 for numbers, small caption below

Visualization: Chart (Pie, Line, or Bar) inside a Paper card

Related Tools: Horizontal ToolCard carousel

Blog Layout

Hero Banner → Blog title + subtitle

Content:

Markdown rendered via MDX

Code blocks styled with Prism.js

Sidebar (desktop): Related articles, ad blocks

Inline ads: after every 2–3 paragraphs

4. Global Features
Dark/Light Mode

Language Support

Language Switcher → Dropdown with flags/icons in navbar & footer

Responsiveness

Navbar → collapses into hamburger menu

Tools grid → 1 per row on mobile, 2–3 on desktop

Inputs & results → stacked on mobile, side-by-side on desktop
